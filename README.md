# Guess the number

## Descriptions:
Welcome to **Guess the Number**, a fast logical and deduction test.

## How to play:
You have 15 attempts to find the hidden number. The game will show your
answer truth level for every submition.

## Supported browsers:
Chrome    : True
Firefox   : True
Explorer  : False

Simple dark designed.
